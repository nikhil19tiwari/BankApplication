## 🚀 Project Overview

This is a **Spring Boot** application simulating an **online banking system**.  
It uses **MySQL** as the database and runs inside Docker containers for easy setup.  

**Features:**
- REST API for bank operations
- Dockerized Spring Boot + MySQL setup
- Actuator endpoints for health and monitoring
- Configurable database and connection pool

---

## 🛠 Prerequisites

Before you start, ensure you have:

- [Docker](https://docs.docker.com/get-docker/) installed  
- [Docker Compose](https://docs.docker.com/compose/install/) installed  
- Git (to clone the repository)
